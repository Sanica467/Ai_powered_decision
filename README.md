# 🚀 DecisionAI – AI Powered Decision Engine

> **Transform Business Data into Intelligent Decisions**

DecisionAI is an AI-powered Business Intelligence platform that acts as an **Autonomous AI Business Analyst**. It enables users to upload business datasets, automatically analyzes the data, detects business issues, predicts future trends using Machine Learning, identifies root causes, and provides actionable business recommendations through an interactive AI interface.

---

## 📌 Problem Statement

Organizations generate large amounts of business data every day, but extracting meaningful insights often requires technical expertise in data science and machine learning.

DecisionAI simplifies this process by allowing users to upload a dataset and receive AI-driven insights without writing any code or selecting machine learning models manually.

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/Sanica467/AI_Powered_Decision_Engine.git
```

### Navigate to the project

```bash
cd AI_Powered_Decision_Engine
```

### Frontend

```bash
npm install
npm run dev
```

### Backend

```bash
cd backend

pip install -r requirements.txt

uvicorn main:app --reload
```
